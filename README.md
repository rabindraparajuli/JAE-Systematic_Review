## [A Global Systematic Review of the Effects of Hydromorphological Floodplain Restoration on Biodiversity](https://doi.org/10.1111/1365-2664.70485)

[![DOI](https://github.com/rabindraparajuli/JAE-Systematic_Review/blob/main/doi-badge-small.svg)](https://doi.org/10.1111/1365-2664.70485)

This repository has data and R codes required to create **Figure 2 | Heatmaps of the proportion of studies achieving qualitative conservation success** of the paper published in _Journal of Applied Ecology_. 

**Files description:**
1. data.xlsx: Excel sheet with data
2. heatmap.R: R codes to summarize, tabulate, visualize and export the heatmap
3. All plots (Heatmap_combined_legend.tif, Intervention_Taxa.tif, and Spatial_scales_Taxa.tif) that were generated using the data and code given
